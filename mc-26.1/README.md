# CreeperSense (Minecraft 26.1)

This folder contains **separate 26.1 builds** for Fabric and NeoForge (26.1 requires Java 25 + Gradle 9+, and the multi-loader toolchain we use for 1.21.1 cannot currently process 26.1 jars).

- `fabric/` — Fabric Loom 1.15 + Fabric API for 26.1
- `neoforge/` — NeoForge MDG 2.0.141 + NeoForge `26.1.0.1-beta`

Both builds share the same `shared/` sources.
